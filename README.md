# 🧠 AstrBot 情绪模拟好感状态插件
> **FavourProPlus** 是一款深度模拟人类心理动力学的高级情感引擎插件。它通过潜意识双引擎架构将短期的 **4D 情绪波动**与长期的 **关系记忆** 结合，辅以独特的“群聊心境传染”、“单峰空闲惩罚”与“指数衰减”机制，赋予 AI 具备环境感知力、物理惯性且极其真实的人格化交互灵魂。

## 🌟 核心特性

### 1. 🎭 潜意识双引擎架构 (Dual-Engine)

- **显意识演绎 (Main LLM)**：专注于角色扮演，通过注入的自然语言心理描述（NLP）来驱动语气和行为，不再输出枯燥的数值标签。
- **潜意识评估 (Unconscious LLM)**：异步运行的“心理分析师”，在后台阅读聊天记录，评估用户的动机、态度并决定情感走向。

### 2. 📊 情感动力学模型

- **4D 短期情绪 (快变量)**：包含 **对他力比多 (亲近欲)**、**对他攻击性 (防备心)**、**自身力比多 (自信)**、**自身攻击性 (负罪感)**。数值随对话实时波动，具有物理惯性。
- **长期记忆 (慢变量)**：包含 **好感度 (Favour)**、**核心印象 (Attitude)** 与 **关系定位 (Relationship)**。好感度建立极其缓慢，遵循非对称增减原则。

### 3. 🌍 全局心境传染 (Global Mood)

- **情绪余波**：在群聊中，AI 的心情会受整体氛围影响。当某人激怒 AI 时，产生的“环境偏移量”会波及到其他群友，模拟真实的人性迁怒。
- **性格捏脸**：支持通过配置全局偏移量预设 AI 的出厂性格（如设定初始防备心偏移为 +15 即可打造“傲娇/暴躁”性格）。

### 4. ⏳ 物理化时间机制

- **指数平滑衰减**：情绪不再突变，而是随时间流逝呈指数级向基线平复（Homeostasis）。
- **单峰空闲惩罚**：模拟“先焦虑、后愤怒、再到冷淡”的心理心路历程。惩罚强度在失联 12 小时左右达到峰值，随后因情感脱敏而转为冷淡。

### 5. 🏷️ 持久化状态追踪

- **意图锁定**：自动捕捉用户的离别意图（如“去睡觉”、“开会中”）。
- **免疫折叠**：状态标签直接持久化于数据库，即使群聊记录因超长被硬折叠/摘要化，AI 依然能精准记住你离开时的状态，并据此判定失联的合理性。

## 🛠️ 运行机制

### 情绪合成公式

$$Effective\_Emotion = Personal\_State + Global\_Offset + Personality\_Preset$$

- **个人状态**：你与 AI 的私交。
- **全局偏移**：当前群聊氛围带来的即时心境。
- **性格预设**：开发者定义的出厂性格底色。

### 情感代谢原则

- **对话回落**：每轮对话情绪会自动向基线代谢 10%，模拟交流过程中的心理适应。
- **极速平复**：群聊产生的环境心境衰减速度远快于个人情感，保证氛围“快聚快散”。

## ⚙️ 配置项说明

- `emotion_sensitivity`: 控制短期情绪受言语刺激影响的强度。
- `turn_homeostasis_rate`: 对话中的情绪代谢率。
- `global_turn_decay_rate`: 群聊氛围的消散速度。
- `idle_threshold_hours`: 触发空闲惩罚的临界时间。
- `decay_duration_hours`: 情绪恢复平静的物理周期。
